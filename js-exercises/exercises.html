Javascript Exercises

Functions

  1. Write a function eleven that returns 11.

    Output:
     eleven() // 11

  2. Write a function twenty that outputs 20.

    Output:
     twenty() // 20

  3. Write a function aString that returns "this is a string"

    Output:
     aString() // "this is a string"

  4. Write a function anArray that outputs the following array: [2, 4, 6, 1]

    Output:
    anArray() // [2, 4, 6, 1]

  5. Write a function identity that receives a value as input and returns it.

    Examples
                      
                    identity(7)       // 7
                    identity("test")  // "test"
                    identity([1,2,3]) // [1,2,3]

  6. Write a function plusTwo that receives a value as input and outputs the value + 2.

    Examples
                      
                    plusTwo(7)       // 9
                    plusTwo(0)       // 2
                    plusTwo(-3)      // -1

  7. Write a function square that receives a number n and returns its square (n2)

    Examples
                      
                    square(7)       // 49
                    square(-3)      // 9
                    square(10)      // 100

  8. Write a function greaterThanTen that receives a number and returns true if the number is greater than 10

    Examples
                      
                    greaterThanTen(5)       // false
                    greaterThanTen(10)      // false
                    greaterThanTen(11)      // true
                    greaterThanTen(15)      // true

  9. Rewrite the function greaterThan without any if statement

  10. Write a function greaterThan that receives a two arguments and returns true if the first argument is greater than the second

    Examples
                      
                    greaterThan(5, 7)       // false
                    greaterThan(9, 9)       // false
                    greaterThan(11, 3)      // true
                    greaterThan(15, -3)     // true

  11. Rewrite the function greaterThanTen using the function greaterThan

  12. Write a function greatestOfTwo that accepts two arguments and returns the greatest of the two

    Examples
                      
                    greatestOfTwo(5, 7)     // 7
                    greatestOfTwo(9, 9)     // 9
                    greatestOfTwo(11, 3)    // 11
                    greatestOfTwo(15, -3)   // 15

  13. Write a function greatestOfFour that receives a four arguments and returns the greatest of the four

    Examples
                      
                    greatestOfFour(5, 7, 9, 1)       // 9
                    greatestOfFour(-7, -6, -5, -3)   // -3
                    greatestOfFour(11, 11, 10, 10)   // 11
                    greatestOfFour(-1, 0, -3, -99)   // 0

  14. Rewrite the greatestOfFour function using the greatestOfTwo function

  15. Write a function countdown that receives a number n. If n <= 0, it should return the string "BOOM!". Otherwise, it should print the number and return countdown(n-1)

    Examples
                      
                    countdown(3)       // prints 3 2 1, returns "BOOM!"
                    countdown(7)       // prints 7 6 5 4 3 2 1, returns "BOOM!"

  16. Write a function sum that receives a number n and returns the sum of the first n numbers

    Examples
                      
                    sum(3)       // 6
                    sum(7)       // 28
                    sum(10)      // 55

  17. Write a function factorial that receives a number n and returns n!. Remember: 0! == 1

    Examples
                      
                    factorial(0)    // 1
                    factorial(3)    // 6
                    factorial(7)    // 5040
                    factorial(10)   // 3628800

  18. Write a function factorialRecursive that receives a number n and returns n! without using any for statements

  19. Write a function newArray that receives a number n and returns an array with n zeros

    Examples
                      
                    newArray(0)    // []
                    newArray(3)    // [0, 0, 0]
                    newArray(7)    // [0, 0, 0, 0, 0, 0, 0]

  20. Write a function buildArray that receives a number n and returns an array containing the first n numbers

    Examples
                      
                    buildArray(0)      // []
                    buildArray(1)      // [0]
                    buildArray(5)      // [0, 1, 2, 3, 4]

  21. Write a function addToArray that receives an array arr and a number n and returns a new array with n appended to arr

    Examples
                      
                    addToArray([1,2,3], 4)          // [1, 2, 3, 4]
                    addToArray([], 8)               // [8]
                    addToArray([5, 9, 72], "str")   // [5, 9, 72, "str"]

  22. Write a function contains that receives an array arr and a number n and returns a true if n is in the array arr

    Examples
                      
                    contains([1,2,3], 2)          // true
                    contains([5, 9, 72], 5)       // true
                    contains([], 8)               // false
                    contains([8, 8, 8, 8], 7)     // false
